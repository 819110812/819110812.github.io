### 方案2： CTF Clinical Trial Processor

是一个软件，需要下载

#### 所用语言

Java

#### 使用方式

1. java jar部署/installer
2. CTF提供一个PIPELINE的功能，将DICOM导入到CTF本地，然后内部去标识化，然后通过HTTP去访问结果



#### 支持的去标识化功能

- dicom object的去标识化
- Dicom 像素的去标识化



#### 评价：

优点：功能很强大，支持很多dicom需要的功能

缺点：部署和使用都需要另外spike去看，部署，未知风险多
